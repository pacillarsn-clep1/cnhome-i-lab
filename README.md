# cnhome-i-lab

## 项目简介

`cnhome-i-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态页面的集中存放与展示空间。

这里收录的页面内容多样，涵盖实验性项目、工具页面、信息展示页等，适合作为个人或小团队的页面存档库。

## 目录说明

```
cnhome-i-lab/
├── pages/          # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/         # 公共资源文件（可选）
│   ├── css/
│   ├── js/
│   └── images/
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/`：主要存放各独立 HTML 页面，每个页面可包含其自身所需的样式与脚本。
- `assets/`：存放多个页面可能共用的资源，如 CSS、JavaScript 或图片文件。
- `index.html`：可选的仓库首页，用于展示页面列表或导航。
